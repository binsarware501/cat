11.6.2024: NDA Repository

Packaging, downloading data, and MINDAR databases are currently unavailable on the NIMH Data Archive (NDA). We do not know when NDA data sharing will resume. Please reach out to the NDA Help Desk at ndahelp@mail.nih.gov for updates.

The ABCD 6.0 data release is expected in January 2025 via the NIH Brain Development Cohorts (NBDC) Data Sharing Platform. Please note that prior data releases previously shared on NDA will be transferred to the NBDC platform. Here is a link to the announcement. More information will be provided soon.

11.6.2024: Data Access Issues

Some data users have encountered issues with their DUC applications or renewals. Our understanding of the situation is that an Executive Order (link here) has been issued by the Biden Administration that prohibits the sharing of Americans’ bulk personal data, including genomic data, in certain cases. Because the ABCD Study includes genomic data along with the rest of our participants’ data in our public data releases, it appears that this executive order is affecting data users from certain countries identified in the Executive Order.

12.13.2023: The ABCD 5.1 Data Release is available on the NIMH Data Archive (NDA).

The 5.1 Patch Release addressed some issues from the 5.0 Data Release. Please click here for further information.

Welcome to the Wiki of the Adolescent Brain Cognitive Development℠ (ABCD) Study!

An overview of the ABCD Study® can be found at abcdstudy.org and detailed descriptions of the assessment protocols are available at ABCD Protocols.

Some users have noted issues in downloading the data, which are addressed in the FAQ. Please refer to the Release Notes on this website for more detailed information.

Please note that NDA no longer requires an NDA Study to be created for public disclosure of results pertaining to ABCD analysis. This will be reinstated in future data releases.

Please visit Now I Know My ABCD for a discussion forum regarding ABCD data. Note that the forum is designed by and for external users to engage in open conversations and exchange ideas related to ABCD data use. The forum serves as a platform for discussions, but the content and opinions expressed may not always be validated or endorsed by ABCD. For more info regarding Now I know My ABCD, please see here.
